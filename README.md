# Drogaria Monteiro - Landing Page

Projeto de uma Landing Page responsiva desenvolvida para a Drogaria Monteiro, com o objetivo de apresentar seus serviços, promoções e facilitar o contato dos clientes através do WhatsApp.

## Funcionalidades

* Página totalmente responsiva
* Seção de apresentação da empresa
* Promoções da semana
* Serviços oferecidos
* Depoimentos de clientes
* Formulário de contato
* Integração com WhatsApp
* Navegação suave entre seções
* Animações ao rolar a página

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Este projeto foi desenvolvido como parte de atividades práticas e para compor meu portfólio de desenvolvimento web, aplicando conceitos de responsividade, estruturação semântica e experiência do usuário.
